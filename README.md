# 🛡️ D&D Party Manager
Мини-проект на React для управления отрядом: добавление, редактирование, удаление персонажей, бросок инициативы, поиск и фильтрация.

## 📥 Установка и запуск
1. Убедитесь, что установлен **Node.js 18+**
2. Откройте терминал в папке проекта
3. Установите зависимости:
   ```bash
   npm install
   npm run dev
   ```
