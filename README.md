# D&D Party Manager
Мини-проект на React для управления отрядом: добавление, редактирование, удаление персонажей, бросок инициативы, поиск и фильтрация.

## Технологии
- **React** (Hooks: `useState`, `useEffect`)
- **Vite** (сборщик и dev-сервер)
- **JavaScript (JSX)**
- **CSS** (Flexbox, Grid, кастомные переменные)
  
## Установка и запуск
1. Убедитесь, что установлен **Node.js 18+**
2. Откройте терминал в папке проекта
3. Установите зависимости:
   ```bash
   npm install
   npm run dev
   ```
