# D&D Party Manager

Мини-проект на React для управления отрядом приключенцев. Позволяет добавлять, редактировать и удалять персонажей, бросать инициативу, а также искать и фильтровать участников группы.

## ✨ Функционал
- ✅ **CRUD операции**: добавление, отображение, редактирование и удаление персонажей
- 🔍 **Поиск и фильтрация**: мгновенный поиск по имени и фильтр по классу
- 🎲 **Бросок инициативы**: генерация случайного числа (d20) и автоматическая сортировка списка по результату
- 📊 **Условный рендеринг**: визуальный статус при HP ≤ 0, динамический текст кнопок, обработка пустого списка
- 🎨 **Стилизация**: тёмная тема, адаптивная вёрстка, удобные компоненты

## 🛠 Технологии
- **React** (Hooks: `useState`, `useEffect`)
- **Vite** (сборщик и dev-сервер)
- **JavaScript (JSX)**
- **CSS** (Flexbox, Grid, кастомные переменные)
